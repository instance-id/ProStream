# Pro Tips

<snippet id="pro_tips">

<chapter title="Pro Tips" id="pro-tips" collapsible="true">
    <tabs>
        <tab title="Start Small">
            <list>
                <li>Test with 20-30 objects first</li>
                <li>Verify everything works</li>
                <li>Then scale up to your actual scene</li>
            </list>
        </tab>
        <tab title="Use Examples">
            <list>
                <li>Enable example rules and modify them</li>
                <li>Don't create rules from scratch initially</li>
                <li>Learn by observing what works</li>
            </list>
        </tab>
        <tab title="Check Console">
            <list>
                <li>Console messages guide you through the process</li>
                <li>Errors are usually clear about what's missing</li>
                <li>Don't ignore warnings</li>
            </list>
        </tab>
        <tab title="Save Often">
            <list>
                <li>Save scene before Calculate Positions</li>
                <li>Save scene before Create SubScenes</li>
                <li>ProStream operations can't be undone easily</li>
            </list>
        </tab>
        <tab title="Iterate">
            <list>
                <li>You can re-run Calculate Positions anytime</li>
                <li>Adjust rules and re-calculate</li>
                <li>Only create SubScenes when satisfied with configuration</li>
            </list>
        </tab>
    </tabs>
</chapter>

</snippet>
