# Common Snippets

This file contains reusable content snippets used across multiple documentation pages.

<snippet id="loading-trigger-definition">

**Loading Trigger:** The GameObject (typically a player or camera) whose position determines which SubScenes load/unload based on distance.

</snippet>

<snippet id="scene-connector-requirement">

> **Prerequisite:** Your scene must have a **SceneConnector** component. See [Scene Connector](scene-connector.md) for details.
>
> {style="note"}

</snippet>

<snippet id="dots-ecs-note">

> ProStream uses Unity's DOTS (Data-Oriented Technology Stack) and ECS (Entity Component System) for high-performance streaming.
>
> {style="note"}

</snippet>

<snippet id="save-before-process">

> **Important:** Save your scene before running this process. ProStream operations cannot be easily undone.
>
> {style="warning"}

</snippet>

<snippet id="console-check">

> Check the **Console** window for detailed progress and any errors.
>
> {style="tip"}

</snippet>

</snippet>
